# GT Racing Game - Unity Project

## Unity Version: 2023.2.20f1
## Rendering Pipeline: Universal Render Pipeline (URP)
## Target Platforms: PC, Mac, Linux, PlayStation 5, Xbox Series X/S

### Project Structure
```
GTRacingGame/
├── Assets/
│   ├── Scripts/
│   │   ├── Car/                    # Car physics and control
│   │   ├── Network/                # Backend communication
│   │   ├── UI/                     # User interface systems
│   │   ├── Multiplayer/            # Real-time multiplayer
│   │   ├── Audio/                  # Audio management
│   │   ├── Telemetry/              # Data collection
│   │   └── Utils/                  # Utility scripts
│   ├── Prefabs/
│   │   ├── Cars/                   # Car prefabs
│   │   ├── UI/                     # UI prefabs
│   │   └── Track/                  # Track elements
│   ├── Materials/                  # Car and track materials
│   ├── Models/                     # 3D models
│   ├── Textures/                   # Texture assets
│   ├── Audio/                      # Audio clips
│   └── Scenes/                     # Game scenes
│       ├── MainMenu.unity
│       ├── Garage.unity
│       ├── Race.unity
│       └── Multiplayer.unity
```
